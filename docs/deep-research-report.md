# 日本のITエンジニア向け主要資格一覧

## エグゼクティブサマリ

ご指定の除外資格を外したうえで、**「日本のITエンジニアが実際に取りに行く主要どころ」**に絞ると、国内国家資格は IPA 上位試験、国際系はクラウド・セキュリティ・ネットワーク・データ・アジャイル・DevOps 系の **14資格**に圧縮するのが実務的です。

## 補足メモ

実務上の注意点として、**Azure ソリューション アーキテクト エキスパート**は前提に **Azure Administrator Associate** があり、**Oracle Certified Java Programmer, Gold SE 17** は **Silver SE 17** など既存パスが関わります。**Certified ScrumMaster** は試験単独型ではなく、公式トレーニング受講を伴う認定です。 citeturn7search5turn28search2turn24search0  

一方で、**Certified Kubernetes Administrator** は実技中心のパフォーマンスベース試験、**PRINCE2 Foundation** は公式教材が試験バウチャーに付属、**CISSP** は日本語の公式ガイドと問題集が整っており、学習導線が比較的明確です。 citeturn17search5turn26search0turn25search2  

**CompTIA Security+** は現行が **V7 / SY0-701** で、日本語試験は 2024年4月23日から提供されています。日本語の古い対策本は試験バージョンが合わないことがあるため、Security+ だけは特に**公式ページ中心**で準備したほうが安全です。 citeturn12search2turn25search0  

データ・AI枠については、G検定/E資格が除外されているため、今回は**「データサイエンス理論の証明」よりも「クラウド上でデータ基盤を設計・運用できることの証明」**に寄せてあります。日本の IT エンジニア向けという条件に対しては、このほうが職種横断で使いやすい構成です。
